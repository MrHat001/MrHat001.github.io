HeartBeat


HeartBeat is an "endless runner" Objective C iOS game. HeartBeat uses a connection to a Bluetooth heart beat sensor to get data from the user, if the player's heart beat is low, it will make the game harder and visa versa.

Pocket Dimension


Pocket Dimension is a flash game where the player must get to the end gate in each level. The player moves and jumps with the arrow keys and can press space bar to use their pocket dimension to place or remove colored blocks.

Stay Alive

Traveling Toast


Traveling Toast is an iPad game created natively in Objective C using SpriteKit. The player plays as a piece of Toast running away from the children trying to eat him for breakfast. The Toast is running through a storm of jelly which slows the him down, and butter slabs which speed the him up. The player can touch the screen to make the Toast to jump up higher to avoid jelly blobs and to run into butter slabs.
